First demo app with Rust language. 

## Inspired by

 https://medium.com/journey-to-rust/drawing-in-gtk-in-rust-part-1-4a401eecc4e0

## Requirements
GTK+ 3.0 installed
    
    brew install gtk+3
    
or 

    sudo apt-get install libgtk-3-dev
